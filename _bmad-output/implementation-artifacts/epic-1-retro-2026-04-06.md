## What Could Be Improved

- **Windows PATH issue** — `uv` not on PATH in bash shell, requiring `python -m uv` throughout. Minor but recurring friction.
- **`__init__.py` export pattern** — initial implementation had `__all__` without actual imports. Caught in code review, but should be established as convention from the start.

## Key Technical Decisions Validated

- **Real `Popen[str]` in MockAgentExecutor** — satisfies mypy --strict while remaining fully configurable. Trivial subprocess (`python -c "..."`) is the right approach.
- **Protocol over ABC** — structural typing works well for AgentExecutor and TaskEventListener. No inheritance needed, mocks just implement the methods.
- **`from __future__ import annotations`** — used consistently for forward references, avoids circular imports with TYPE_CHECKING guard.
